# Sunshine Book Store

Sunshine Book Store is a Spring Boot application designed to manage book information. Users can perform various operations such as adding books, deleting books, adding books to the cart, updating book information, and more.

## Features

- User Registration: Users can register themselves to access the bookstore functionalities.
- User Authentication: Registered users can log in using their credentials.
- Password Management: Users can change their passwords for security purposes.
- Book Operations: Users can add, delete, update, and view book information.
- Cart Management: Users can add books to their cart for future purchase.
- Logout: Users can securely log out of the application.

## Technologies Used

- Java
- Spring Boot
- Spring Security
- Thymeleaf (for front-end templating)
- MySQL (for data storage)
- Maven (for dependency management)

## Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/your-username/sunshine-book-store.git
```
2. Navigate to the project directory:
```bash
cd sunshine-book-store
```
3. Set up MySQL database:
> Create a MySQL database and configure the database credentials in `application.properties`.

4. Build and run the application using Maven:
```bash
mvn spring-boot:run
```
5. Access the application in your web browser at `http://localhost:8080`.

## Usage
- Register a new user by providing necessary details.
- Log in with your credentials.
- Explore the bookstore functionalities such as adding, deleting, updating books, managing cart, etc.
- Change your password if needed.
- Log out securely after completing your tasks.
